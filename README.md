# PaLM-Backend


## Getting Started

Follow these steps to get started with the project:


- Go to project directory
```bash
cd backend
```

- Edit .env file
```bash
secretkey=your_secret_key
url=mongodb_database_url
```
- Install Node Packages
```bash
npm install 
```

- Run the server
```bash
npm run dev
```
